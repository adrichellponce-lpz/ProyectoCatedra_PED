# Proyecto Cátedra PED - Fase 2

## Descripción
Dashboard principal del programa e implementación de TAD.

## Instalación
1. Clona el repositorio dentro de visual estudio.
2. Abre el archivo `ProyectoCatedra_PED.sln` en Visual Studio.
3. Asegúrate de tener instalado el SDK de .NET.
4. Recompilar la solución para evitar errores de referencias faltantes.

## Ejecución
1. Compila la solución.
2. Presiona `F5` o el botón "Iniciar" en Visual Studio para ejecutar el dashboard.
